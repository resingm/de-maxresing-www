Project Overview

yacf - Yet Another Configuration Framework

My growing involvement in Python based services revealed that I tend to parse configuration files similarly over different projects. Thus, I decided to package and publish my own configuration parsing framework.

The feature scope is quite limited, since it was intended to be a slim framework. Still, it gives the developer the power to merge and prioritize configurations from different sources, e.g. default configuration, custom configuration file, environment variables and command line arguments. The framework reflects my preferred way of handling configurations.
